Seoro 플랫폼 PRD (Product Requirements Document)

1. 트렌드 이슈 해설 기능 (1순위)
- 목적: 한국에서 실시간으로 뜨는 이슈를 수집하고, AI가 문화적/역사적으로 해설해주는 기능 제공
- 세부 구성:
  * 이슈 수집: Firecrawl로 Google Trends 등에서 핫 키워드 자동 수집 (주기적)
  * AI 해설: OpenAI/Claude API로 키워드별 해설 생성 (문화적 맥락, 배경지식 포함)
  * 지도 연동: Kakao Map API로 이슈 관련 장소 검색/표시, 지도 위에 핀+말풍선 정보 노출
  * 유저 상호작용: 댓글, 좋아요/싫어요/리트윗, 이슈 공유

2. 여행 후기 공유 기능
- 목적: 사용자가 여행 중 방문한 장소, 감상, 팁 등을 공유하고 지도에 반영
- 세부 구성:
  * 후기 작성 폼: 위치(지도 클릭/검색), 사진 업로드(Firebase Storage), 텍스트
  * 후기 리스트: 최신순/지역순 정렬
  * 지도 핀 연동: 후기 작성 시 Kakao 지도에 핀 등록, hover 시 요약 미리보기

3. 주간 핫플레이스 추천 기능
- 목적: 매주 한국에서 가장 주목받는 장소를 데이터 기반으로 추천
- 세부 구성:
  * 데이터 수집: Google Trends, 인스타 해시태그 분석
  * 핫플 선정: 음식/카페/관광지 등 카테고리별 분석
  * 지도 연동: 선정 장소 지도 핀+카드형 요약

4. AI 여행 플래너 기능 (후순위)
- 목적: 명소 선택 시 AI가 일정/루트 자동 생성
- 세부 구성:
  * 한국관광공사 API: 지역/테마별 명소 리스트
  * 명소 선택 UI: 유저 선택 기반
  * OpenAI API: 일정+루트 제안
  * 지도 시각화: Kakao Map API로 동선 표시, 플랜 저장/공유

5. 트립메이트 매칭 기능 (후순위)
- 목적: 여행 메이트 매칭 (데이팅앱 스타일)
- 세부 구성:
  * 프로필 기반 매칭 카드(Bumble 스타일)
  * 관심 지역/언어/여행 스타일 기반 추천
  * 상호 매칭 시 1:1 채팅(Firebase Chat)
  * 필터: 날짜/장소/성별/연령
  * 사용자 기반 확보 후 도입

6. 기타 핵심 기능
- 로그인/회원가입: Firebase Auth (Google OAuth)
- 마이페이지: 내가 쓴 후기/좋아요/매칭 히스토리 관리
- 다국어 지원: 영어 UI 기본, 한국어 전환, OpenAI 자동 번역 확장 가능
