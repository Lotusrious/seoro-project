# Task ID: 4
# Title: 지도 시각화 연동 (웹)
# Status: pending
# Dependencies: 1, 3
# Priority: high
# Description: Tmap, KakaoMap 또는 Google Maps API 등을 사용하여 웹 기반 지도 시각화를 구현합니다.
# Details:
1. 기능 및 성능, 웹 적합성을 기준으로 지도 API(Tmap, KakaoMap, Google Maps 등)를 평가하고 선택합니다.
2. 선택한 지도 API를 React 앱에 통합합니다.
3. 지도에 위치에 대한 핀(마커) 배치를 구현합니다.
4. 다양한 유형의 위치에 대한 사용자 지정 마커를 만듭니다.
5. 정보 창(인포윈도우)을 표시하는 마우스 오버/클릭 기능을 개발합니다.
6. 리뷰 및 사진을 표시하는 클릭 기능을 구현합니다.
7. 확대/축소 및 이동 컨트롤을 추가합니다.
8. 사용자의 현재 위치에 지도를 중앙에 표시하는 브라우저 지오로케이션 기능을 활용합니다.
9. 지도 렌더링 성능을 최적화합니다.

# Test Strategy:
1. 다양한 위치에 대한 핀의 올바른 배치를 확인합니다.
2. 마우스 오버/클릭 기능을 테스트합니다.
3. 지도 컨트롤의 반응성을 확인합니다.
4. 지오로케이션 정확도를 테스트합니다.
5. 많은 수의 마커로 성능 테스트를 수행합니다.
