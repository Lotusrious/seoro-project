# Task ID: 5
# Title: 주간 핫플레이스 추천 기능
# Status: pending
# Dependencies: 1
# Priority: medium
# Description: Google Trends, 인스타그램 해시태그 등 다양한 데이터를 분석해 카테고리별로 주간 핫플레이스를 추천하고 지도에 표시합니다.
# Details:
Firecrawl로 트렌드 데이터를 수집하고, 인스타그램 해시태그를 분석해 인기 장소를 선정합니다. 카카오맵에 핀과 카드형 요약으로 표시합니다.

# Test Strategy:
추천된 장소의 정확성과 관련성이 높은지, 지도에 올바르게 표시되는지 확인합니다.
