# Task ID: 13
# Title: 다양한 사용자 유형을 위한 웹 사용자 흐름 개발
# Status: pending
# Dependencies: 11, 12
# Priority: medium
# Description: 초보자, 숙련된 사용자 및 처음부터 코스를 만들고 싶은 사용자를 위한 분기별 사용자 흐름을 웹에 구현합니다.
# Details:
1. 사용자 유형 선택 인터페이스를 웹 페이지로 설계하고 구현합니다.
2. 고정된 코스 옵션이 있는 "초보자" 흐름을 개발합니다.
3. 코스 편집을 허용하는 "숙련된 사용자" 흐름을 만듭니다.
4. 프롬프트 기반 생성을 사용하는 "처음부터 만들기" 흐름을 구현합니다.
5. 숙련된 사용자 및 코스 생성자를 위한 코스 편집 인터페이스를 웹 페이지에 개발합니다.
6. 코스 경유지 순서 변경을 위한 드래그 앤 드롭 기능을 구현합니다 (예: react-beautiful-dnd 등 웹 라이브러리 사용).
7. 사용자 지정 생성 코스를 저장하는 시스템을 만듭니다.
8. 코스 편집을 위한 실행 취소/다시 실행 기능을 구현합니다.
9. 사용자 생성 코스를 위한 미리보기 모드를 개발합니다.

# Test Strategy:
1. 사용자 유형 선택에 따른 올바른 분기를 확인합니다.
2. 편집 인터페이스의 기능과 사용성을 테스트합니다.
3. 코스 경유지의 드래그 앤 드롭 순서 변경을 확인합니다.
4. 사용자 지정 생성 코스의 저장 및 로딩을 테스트합니다.
5. 코스 편집에서 실행 취소/다시 실행 기능을 확인합니다.
6. 각 사용자 유형 흐름에 대한 사용성 테스트를 수행합니다.
7. 미리보기 모드의 정확성을 테스트합니다.
