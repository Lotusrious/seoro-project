# Task ID: 12
# Title: 코스 추천 표시 웹 UI 구현
# Status: pending
# Dependencies: 3, 4, 11
# Priority: high
# Description: AI 생성 코스 추천을 사용자에게 웹 페이지에 표시하는 UI 및 기능을 개발합니다.
# Details:
1. 추천 코스를 표시하기 위한 카드 기반 UI를 웹 페이지로 설계하고 구현합니다.
2. 여러 추천을 위한 캐러셀 또는 스크롤 가능한 목록을 만듭니다.
3. 각 추천에 대한 "세부 정보 보기" 기능을 구현합니다.
4. 지도에 전체 코스를 보여주는 상세 보기 페이지 또는 모달을 개발합니다.
5. 사진 및 간략한 설명을 포함하여 장소 정보를 표시하는 구성 요소를 만듭니다.
6. 추천을 조정하거나 구체화하는 기능을 구현합니다.
7. 추천 코스를 저장하거나 공유하는 옵션을 추가합니다.
8. 성능을 위해 이미지 로딩(lazy loading 등) 및 캐싱을 최적화합니다.
9. 추천과의 사용자 상호 작용을 추적하기 위한 분석 기능을 구현합니다.

# Test Strategy:
1. 코스 추천이 올바르게 표시되는지 확인합니다.
2. 다양한 장치 및 브라우저에서 캐러셀/목록의 반응성을 테스트합니다.
3. 상세 보기의 정확성과 완전성을 확인합니다.
4. 코스 조정 기능의 기능을 확인합니다.
5. 저장 및 공유 옵션을 테스트합니다.
6. 특히 이미지 로딩에 대한 성능 테스트를 수행합니다.
7. 사용성 통찰력을 위해 사용자 상호 작용 데이터를 분석합니다.
