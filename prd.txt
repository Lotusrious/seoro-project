⚙️ 핵심 기능 목록 (전체 요약)
🔍 1. AI 여행 코스 생성
키워드/지역/사용자 조건 기반 코스 추천
장소 간 이동 경로 자동 계산
LLM: GPT-4o (추천/표현), Gemini (요약/분류)

🗺️ 2. 지도 기반 시각화
Tmap 또는 KakaoMap API 연동
장소 핀 표시 + hover 시 인포윈도우
click 시 리뷰/사진 열람

💬 3. 리뷰 자동 수집
출처	방식
구글 플레이스	공식 API
네이버 블로그	Firecrawl + Puppeteer
인스타그램	Apify 해시태그 크롤링
→ 수집 후 GPT 요약 + 팁 생성 → Firebase 저장

🧃 4. 코스 저장 + 마이페이지
사용자가 만든 코스 저장 가능 (LLM 추천 or 직접 제작 포함)
장소 카드, 지도 스크린샷 자동 생성 (html2canvas 등)
마이페이지에서 내가 만든 코스 한눈에 열람 가능

🖼️ 5. 소셜 공유
저장된 코스를 인스타처럼 사진 카드로 공유 가능
공개/비공개 설정
다른 사용자도 좋아요 / 댓글 가능
유저가 다녀온 실제 사진도 첨부 가능

👫 6. 여행 메이트 매칭
코스를 "예정된 여행"으로 등록 → 메이트 모집 가능
모집 조건 설정 (성별, 나이대, 언어 등)
매칭 방식: 키워드 기반 or 위치 기반 중 선택 가능
하루 3명까지 상세 보기 가능 (초과 시 유료)
